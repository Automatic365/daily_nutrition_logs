# Daily Log — Long Term Memory

## Purpose
This file serves as the persistent memory repository for the Combat Nutrition Coach. It maintains continuity across chat sessions by capturing daily measurements, adherence scores, key discussions, and decisions.

**Trigger Behaviors:**
- **"Goodnight"** → Summarize the day's conversation and append a new entry
- **"Good morning"** → Reference recent entries to establish context and acknowledge continuity

**Entry Order:** Chronological (oldest first)
